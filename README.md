# D2C-Skincare-Full-Funnel-Project
Complete full-funnel marketing strategy for a D2C skincare brand including TOFU, MOFU, BOFU execution, performance metrics, and ROI analysis.
## Project Overview
This project demonstrates a complete full-funnel marketing strategy designed for a D2C skincare brand.

## Objectives
- Build brand awareness
- Generate qualified leads
- Increase conversion rate
- Improve customer retention

## Funnel Breakdown

### 1. Top of Funnel (TOFU)
- Social media awareness campaigns
- Influencer collaborations
- SEO blog strategy

### 2. Middle of Funnel (MOFU)
- Retargeting ads
- Email nurturing campaigns
- Lead magnets

### 3. Bottom of Funnel (BOFU)
- Conversion-focused ad creatives
- Limited time offers
- Optimized landing pages

## Tools Used
- Google Analytics
- Meta Ads Manager
- SEO Keyword Tools
- Excel for ROI calculations

## Key Results (Hypotical Case Study)
- CTR improved by 3.2%
- Conversion rate increased by 18%
- CAC reduced by 12%
